# About:
Penumbra is a multi-threaded Luau graphics rendering pipeline, built on the platform-agnostic Pajamas runtime.

Penumbra by default is configured to run a series of GLSL shaders ported to Luau, showcasing its abilities.

# Requirements:
Penumbra requires a valid Pajamas runtime setup, including the following list of plugin modules:
- display
- event
- input
- task
- vm

You can pull Pajamas' files from this repo: https://github.com/Crazyblox/Pajamas
